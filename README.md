# react-skeleton

React プロジェクト用の skeleton.
react-roterでSPAをしつつ、expressでserver-side-rendering.

## Running

```
npm install
npm start
```

## npm packages

- react
- react-dom
- react-router
- babel-cli
- babel-preset-es2015
- babel-preset-react
- webpack 
- webpack-dev-server
- babel-loader

### for server-side-rendering
- express
- path
- compression
- json-loader
